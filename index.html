จากไอเดียไปจนถึงดีไซน์ขึ้นร้าน — ใช้งานได้จริงในทุกขั้นของเวิร์กโฟลว์

TAB 1🔍

Research & Prompt

Date Strip + Trending Niches + AI Prompt ครบในแท็บเดียว

  • Date Strip — วันที่, เดือน, ฤดูกาล USA, Event ที่กำลังมา
  • เลือกเดือน 12 เดือน — ข้อมูลอัปเดตตาม
  • Niche Tags — Seasonal + Evergreen
  • AI Design Prompt สำหรับ Midjourney / DALL·E / Ideogram
  • Trending Niches พร้อม Heat Indicator
  • Event Calendar USA — เพิ่ม Niche ได้
  • Pro Tips ประจำเดือน
TAB 2📝

SEO Listing Generator

สร้าง Listing ครบตามกฎ Amazon Merch

  • กรอก Niche, ประเภทสินค้า, กลุ่มเป้าหมาย, Occasion
  • AI สร้าง Brand / Title / Bullet 1-2 / Description ครบตามกฎ
  • นับตัวอักษร Real-time มีสีเตือน
  • แสดง SEO Keywords ที่ใช้
  • Copy แยก + Copy All
TAB 3🖼️

Image to Listing

มีแค่รูป — AI เขียน Listing ให้อัตโนมัติ

  • อัปโหลดรูป Design → AI อ่านรูป → สร้าง SEO Listing
  • แสดง Detected Niche + Top Keywords
  • สร้าง Design Prompt แตกเพิ่ม
  • เพิ่ม 3 Variations จาก Niche เดียวกัน
TAB 4🚀

Expand Prompts

มี Niche แล้ว ขยายเป็นดีไซน์หลายแบบในธีมเดียว

  • ป้อน Niche → AI สร้าง 3–7 Prompts ในธีมเดียว
  • เลือก Mixed Styles หรือสไตล์เดียวกัน
  • Copy แยก หรือ Copy All
TAB 5 NEW🔬

Design Analyzer

ตรวจก่อนอัปขึ้น Merch — ขายได้ไหม ยิงแอดดีไหม

  • AI ให้คะแนน 10 มิติ (Design, Niche, Competition, SEO ฯลฯ)
  • Sell Verdict YES / MAYBE / NO พร้อม Confidence %
  • Ad Recommendation — Budget, Targeting, Expected ACoS
  • Keyword Table + Volume + Competition
  • คำแนะนำปรับปรุง 4 ข้อ
TAB 6 NEW⚙️

Settings

เชื่อม API Pro หรือใช้ฟรีก็ได้

  • ใส่ API Key ของ Merch Informer, Helium 10 (ถ้ามี)
  • เก็บใน localStorage เครื่องคุณเท่านั้น
  • ไม่มี API Key ก็ใช้งานได้ทุกฟีเจอร์
